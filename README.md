Arjun Manikyath - IA08 DoodleApp

Overview
  - DoodleApp is a interactive drawing application built for Android using Jetpack Compose. It allows users to sketch freely on a canvas with customizable brush settings.
  - The app consists of two main parts: a tool panel where the user can adjust brush settings and clear the canvas, and a drawing surface implemented with Composeâ€™s Canvas and gesture APIs.
Features
    - Tool Panel
        Adjust brush size with a slider
        Choose brush color from a palette
        Select brush style: solid or dotted
        Undo last stroke
        Clear entire canvas
    - Drawing Canvas
        Responsive touch and drag gesture handling
        Real-time stroke rendering using Compose's Canvas
        State-driven UI updates via mutableStateListOf
    - Bonus Feature
        Brush pattern selector
        Undo functionality

Installation Steps
- Open Android Studio to the projects screen
- Click Clone Repository
- Enter the following into the URL field: https://github.com/arjunman0413/ArjunDoodleApp.git
- Choose a Directory of your preference
- Select Clone button in bottom right
- Trust the project
- File --> Sync Project With Gradle Files
- Build --> Assemble Project
- Run Project
